Aplikacja: "Moje Notatki"

opis:

Napisana w czystym JS (składnia ES6). Użyto modułu NPM 'Moment'. Wersja produkcyjna kodu podzielona na moduły, skompilowane końcowo przy użyciu Webpack. Dane użytkownika zapisują się w local-storage.

Aplikacja: "Hangman"

opis:

Aplikacja wykonana w czystym JS (składnia ES6). Kod wersji produkcyjnej został podzielona na moduły, później skompilowany dla wersji oficjalnej przy użyciu Webpack. Jest to wersja angielska popularnej gry "Wisielec", ponieważ korzysta z API napisanego w tymże języku. Autorem API jest Andrew Mead - doskonały wykładowca i autor kursów na Udemy.
